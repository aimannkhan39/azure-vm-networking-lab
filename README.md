# Azure Virtual Machine Networking Lab

## Introduction

This project demonstrates communication between two Microsoft Azure Virtual Machines within the same Virtual Network (VNet) using private IP connectivity.

## Technologies Used

* Microsoft Azure
* Virtual Machines
* Virtual Network (VNet)
* Network Security Groups (NSG)

## Steps Performed

1. Created a Virtual Network
2. Created two Virtual Machines
3. Connected both VMs to the same subnet
4. Verified connectivity using ping command
5. Tested communication between VMs
